**作为** 开发者

## ✅ 验收标准 (BDD 格式)

### AC 1.1: 新增 11 个字段到 users 表

**Given** 当前 User 模型只有 6 个档案字段  
**When** 执行数据库迁移  
**Then** 新增 11 个字段到 users 表:
- `current_weight` (当前体重，单位：克)
- `waist_circumference` (腰围，单位：厘米)
- `hip_circumference` (臀围，单位：厘米)
- `body_fat_percentage` (体脂率，单位：百分比)
- `muscle_mass` (肌肉量，单位：克)
- `bone_density` (骨密度，单位：克/平方厘米)
- `metabolism_rate` (基础代谢率，单位：kcal/day)
- `health_conditions` (健康状况，JSON 格式)
- `medications` (用药情况，JSON 格式)
- `allergies` (过敏信息，JSON 格式)
- `sleep_quality` (睡眠质量，1-10 分)

### AC 1.2: 向后兼容性保证

**Given** User 模型扩展完成  
**When** 现有用户访问档案  
**Then** 新增字段 nullable=True 或 default 值  
**And** 现有用户数据不受影响  

### AC 1.3: API 返回完整字段

**Given** User 模型扩展完成  
**When** API GET /users/me 被调用  
**Then** 返回所有 17 个字段  
**And** 新增字段对于现有用户为 null  

### AC 1.4: 重量单位一致性

**Given** 重量相关字段  
**When** 存储到数据库  
**Then** 统一使用克 (grams) 为单位  
**And** current_weight, muscle_mass 等字段遵循重量单位约定  

### AC 1.5: 数据验证规则

**Given** 新增字段有验证规则  
**When** 用户提交数据  
**Then** 验证规则正确执行:
- age: 0-120
- height: 50-250 cm
- weight 字段: 20000-300000 g (20-300kg)
- body_fat_percentage: 3-70%
- sleep_quality: 1-10

import pytest
from alembic.config import Config
from alembic import command

class TestUserMigration:
    """测试 User 模型迁移"""
    
    def test_migration_upgrade(self, test_engine):
        """测试迁移可以成功升级"""
        # Given: Alembic 配置
        alembic_cfg = Config("alembic.ini")
        
        # When: 执行迁移
        command.upgrade(alembic_cfg, "head")
        
        # Then: 新列存在
        from sqlalchemy import inspect
        inspector = inspect(test_engine)
        columns = [col['name'] for col in inspector.get_columns('users')]
        
        assert 'current_weight' in columns
        assert 'waist_circumference' in columns
        assert 'body_fat_percentage' in columns
        # ... 所有新字段
    
    def test_migration_downgrade(self, test_engine):
        """测试迁移可以成功回滚"""
        # Given: 已升级到 head
        alembic_cfg = Config("alembic.ini")
        command.upgrade(alembic_cfg, "head")
        
        # When: 回滚一个版本
        command.downgrade(alembic_cfg, "-1")
        
        # Then: 新列不存在
        from sqlalchemy import inspect
        inspector = inspect(test_engine)
        columns = [col['name'] for col in inspector.get_columns('users')]
        
        assert 'current_weight' not in columns
        assert 'waist_circumference' not in columns
