## ✅ 验收标准 (BDD 格式)

### AC 1.1: UserGoal 表创建

**Given** 需要目标追踪  
**When** 创建新表  
**Then** 表结构包含:
- goal_id (主键)
- user_id (外键，关联 users 表)
- goal_type (目标类型：weight/exercise/diet/habit)
- current_value (当前值)
- target_value (目标值)
- unit (单位)
- start_date (开始日期)
- target_date (目标日期)
- predicted_date (预测达成日期)
- status (状态：active/completed/paused/cancelled)

### AC 1.2: GoalProgress 表创建

**Given** UserGoal 表已创建  
**When** 追踪目标进度  
**Then** GoalProgress 表结构:
- progress_id (主键)
- goal_id (外键，关联 user_goals 表)
- recorded_date (记录日期)
- value (值)
- daily_target_met (是否达成每日目标)
- streak_count (连续达成天数)

### AC 1.3: GoalHistory 表创建

**Given** 目标变更需要追溯  
**When** 目标状态变更  
**Then** GoalHistory 表结构:
- history_id (主键)
- goal_id (外键)
- change_type (变更类型)
- previous_state (JSON，变更前状态)
- new_state (JSON，变更后状态)
- reason (变更原因)
- ai_suggested (是否 AI 建议)

### AC 1.4: 数据库迁移脚本

- 创建 3 个新表

### AC 1.5: SQLAlchemy 模型

### 数据模型设计

```python
from sqlalchemy import Column, Integer, String, Float, DateTime, ForeignKey, Boolean, Text
from sqlalchemy.dialects.postgresql import JSON
from sqlalchemy.orm import relationship
from sqlalchemy.sql import func
from enum import Enum

class GoalType(str, Enum):
    WEIGHT = "weight"
    EXERCISE = "exercise"
    DIET = "diet"
    HABIT = "habit"

class GoalStatus(str, Enum):
    ACTIVE = "active"
    COMPLETED = "completed"
    PAUSED = "paused"
    CANCELLED = "cancelled"

class UserGoal(Base):
    __tablename__ = "user_goals"
    
    goal_id = Column(Integer, primary_key=True, index=True)
    user_id = Column(Integer, ForeignKey("users.id"), nullable=False, index=True)
    goal_type = Column(String(50), nullable=False, index=True)  # GoalType
    current_value = Column(Float, nullable=True)
    target_value = Column(Float, nullable=False)
    unit = Column(String(20), nullable=False)  # kg, steps, kcal, etc.
    start_date = Column(DateTime(timezone=True), server_default=func.now())
    target_date = Column(DateTime(timezone=True), nullable=True)
    predicted_date = Column(DateTime(timezone=True), nullable=True)
    status = Column(String(20), default="active")  # GoalStatus
    
    # 关系
    user = relationship("User", back_populates="goals")
    progress = relationship("GoalProgress", back_populates="goal", cascade="all, delete-orphan")
    history = relationship("GoalHistory", back_populates="goal", cascade="all, delete-orphan")

class GoalProgress(Base):
    __tablename__ = "goal_progress"
    
    progress_id = Column(Integer, primary_key=True, index=True)
    goal_id = Column(Integer, ForeignKey("user_goals.goal_id"), nullable=False, index=True)
    recorded_date = Column(DateTime(timezone=True), server_default=func.now(), index=True)
    value = Column(Float, nullable=False)
    daily_target_met = Column(Boolean, default=False)
    streak_count = Column(Integer, default=0)
    
    # 关系
    goal = relationship("UserGoal", back_populates="progress")

class GoalHistory(Base):
    __tablename__ = "goal_history"
    
    history_id = Column(Integer, primary_key=True, index=True)
    goal_id = Column(Integer, ForeignKey("user_goals.goal_id"), nullable=False, index=True)
    change_type = Column(String(50), nullable=False)  # created, updated, completed, paused, etc.
    previous_state = Column(JSON, nullable=True)
    new_state = Column(JSON, nullable=False)
    reason = Column(Text, nullable=True)
    ai_suggested = Column(Boolean, default=False)
    created_at = Column(DateTime(timezone=True), server_default=func.now())
    
    # 关系
    goal = relationship("UserGoal", back_populates="history")
```

        assert goal.status == GoalStatus.ACTIVE
    
    def test_goal_relationships(self, db_session, test_user):
        """测试目标关系"""
        # Given: 创建目标和进度
        goal = UserGoal(
            user_id=test_user.id,
            goal_type=GoalType.WEIGHT,
            target_value=65.0,
            unit="kg",
        )
        progress = GoalProgress(
            goal_id=goal.goal_id,
            value=69.5,
            daily_target_met=True,

        db_session.add(progress)
        db_session.commit()

        assert test_user.goals == [goal]
        assert goal.progress == [progress]
        assert progress.goal == goal
