## 1. 需求概述

### 1.1 目标

构建一个统一的用户记忆系统，涵盖用户的短期会话上下文和长期画像记忆，支持个性化服务和语义搜索能力。

### 1.2 需求来源

本需求来源于产品定义阶段的核心目标：让所有用户行为进入 agent 记忆，包括运动打卡、饮食记录等，为后续的主动触达和个性化服务提供数据基础。

### 1.3 功能需求

| 需求编号 | 需求描述 | 优先级 |
|----------|----------|--------|
| MEM-001 | 支持用户打卡数据（运动、饮食）的持久化存储 | P0 |
| MEM-002 | 支持跨会话的长期用户画像存储 | P0 |
| MEM-003 | 支持习惯模式检测和记忆 | P0 |
| MEM-004 | 支持用户偏好和目标的存储 | P0 |
| MEM-005 | 支持语义搜索能力（向量相似度匹配） | P1 |
| MEM-006 | 支持记忆分层：短期（原始数据）→ 长期（索引后） | P0 |
| MEM-007 | 短期记忆保留周期可配置（默认7天） | P0 |
| MEM-008 | 统一记忆模型，不按角色分离 | P0 |

### 1.4 非功能需求

| 需求编号 | 需求描述 | 优先级 |
|----------|----------|--------|
| NFR-001 | 向量嵌入使用本地模型，预留云端切换接口 | P0 |
| NFR-002 | 配置存储在 config.yaml | P0 |
| NFR-003 | 支持定时任务执行记忆索引 | P0 |

### 3.1 统一记忆模型

统一记忆表（unified_memory）是长期记忆的核心存储，取代原有的 user_long_term_memory 表。

```python
class UnifiedMemory:
    """统一记忆模型"""
    
    # 基础字段
    id: str                    # UUID 主键
    user_id: str               # 用户ID
    
    # 记忆内容
    memory_type: str           # 记忆类型
                               # - "打卡_pattern"    : 打卡行为模式
                               # - "偏好_inferred"   : 推断的偏好
                               # - "目标_explicit"   : 明确的目标
                               # - "习惯_completed"  : 习惯完成记录
                               # - "里程碑_achieved" : 成就里程碑
                               # - "趋势_insight"   : 数据趋势洞察
                               # - "关联_causal"    : 因果关联
    content_raw: str           # 原始内容（可选）
    content_summary: str       # LLM 摘要（核心内容）
    content_keywords: list     # 关键词（可检索）
    
    # 向量存储（pgvector）
    embedding: Vector(768)     # 向量嵌入
    
    # 来源信息
    source_type: str           # 来源类型
                               # - "habit_record"   : 习惯记录
                               # - "conversation"  : 对话
                               # - "health_record"  : 健康记录
                               # - "meal_record"    : 饮食记录
                               # - "exercise_record": 运动记录
    source_id: str             # 来源数据ID（可追溯）
    
    # 元数据
    importance_score: float    # 重要性评分 (0-10)
    created_at: datetime       # 创建时间
    updated_at: datetime       # 更新时间
    last_accessed: datetime   # 最后访问时间
```

```python
class MemoryIndexPipeline:
    """记忆索引 Pipeline"""
    
    def __init__(
        self,
        extractor: MemoryExtractor,
        embedding_provider: EmbeddingProvider,
        db_session: Session
    ):
        self.extractor = extractor
        self.embedding_provider = embedding_provider
        self.db = db_session
    
    async def run(self, user_id: str = None):
        """
        执行索引 Pipeline
        
        Args:
            user_id: 指定用户ID（可选，不指定则处理所有用户）
        """
        # Step 1: 收集待索引数据
        source_records = await self._collect_pending_records(user_id)
        
        # Step 2-5: 处理每条记录
        for record in source_records:
            try:
                await self._process_record(record)
            except Exception as e:
                await self._log_failure(record, str(e))
    
    async def _collect_pending_records(self, user_id: str = None) -> list:
        """收集待索引的记录"""
        # TODO: 查询未索引的原始记录
        # - habits 表中 is_indexed = False 的记录
        # - health_records 表中 is_indexed = False 的记录
        # - conversations 表中 is_indexed = False 的记录
        pass
    
    async def _process_record(self, record: dict):
        """处理单条记录"""
        # Step 2: 提取记忆
        extracted = self.extractor.extract_from_habit(record)
        
        # Step 3: 生成向量
        embedding = self.embedding_provider.embed(extracted['summary'])
        
        # Step 4: 存储

            user_id=record['user_id'],
            memory_type=extracted['memory_type'],

            content_summary=extracted['summary'],
            content_keywords=extracted['keywords'],
            embedding=embedding,
            source_type=record['type'],
            source_id=record['id'],
            importance_score=extracted['importance']
        )
        self.db.add(memory)
        
        # Step 5: 记录日志
        await self._log_success(record, memory.id)
```

### 4.4 语义搜索服务

```python
class SemanticSearchService:
    """语义搜索服务"""
    
    def __init__(
        self,
        embedding_provider: EmbeddingProvider,
        db_session: Session
    ):
        self.embedding_provider = embedding_provider
        self.db = db_session
    
    async def search(
        self,
        user_id: str,
        query: str,
        memory_types: list = None,
        limit: int = 10
    ) -> list:
        """
        语义搜索用户记忆
        
        Args:
            user_id: 用户ID
            query: 查询文本
            memory_types: 记忆类型过滤（可选）
            limit: 返回数量限制
            
        Returns:
            记忆列表（按相似度排序）
        """
        # 1. 将查询文本向量化
        query_embedding = self.embedding_provider.embed(query)
        
        # 2. 构建查询
        base_query = """
            SELECT id, user_id, memory_type, content_summary, 
                   source_type, source_id, importance_score, created_at,
                   (embedding <=> :query_embedding) as similarity
            FROM unified_memory
            WHERE user_id = :user_id 
              AND is_active = true 
              AND is_indexed = true
        """
        
        params = {
            'query_embedding': query_embedding,
            'user_id': user_id
        }
        
        # 3. 添加类型过滤
        if memory_types:
            placeholders = ', '.join([f':type_{i}' for i in range(len(memory_types))])
            base_query += f" AND memory_type IN ({placeholders})"
            for i, mem_type in enumerate(memory_types):
                params[f'type_{i}'] = mem_type
        
        # 4. 排序并限制
        base_query += " ORDER BY similarity ASC LIMIT :limit"
        params['limit'] = limit
        
        # 5. 执行查询
        result = await self.db.execute(text(base_query), params)
        return result.fetchall()
```
