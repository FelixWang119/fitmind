本 Sprint 实现"每日科普"功能，通过 AI 生成的健康知识卡片帮助用户理解"为什么"，增强用户信任感和专业度。

## 背景说明

### 产品决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 内容来源 | AI 创作 | 成本低、可个性化、规模大 |
| 内容模式 | 混合模式 | 每周主题日 + 用户行为触发 |

### Story 9.1: 科普内容生成服务
**故事点**: 5 pts  
**优先级**: P0

**作为** 系统  
**我想要** 每天自动生成科普内容  
**以便** 为用户提供新鲜的知识点

**验收标准**:
- [ ] 每日凌晨生成当日科普内容
- [ ] 每周一个主题（营养/运动/睡眠/心理）
- [ ] 内容长度：摘要 50 字内，正文 300-500 字
- [ ] 添加医学免责声明
- [ ] 内容存储到数据库，支持历史查看
- [ ] 支持手动触发重新生成

**技术任务**:
- [ ] 创建科普内容数据模型 DailyTip
- [ ] 实现 AI 生成 Prompt 模板
- [ ] 创建定时生成任务
- [ ] 实现内容存储和查询 API
- [ ] 添加免责声明

**依赖**: 无
