```typescript
  }
};
```

### 日历生成逻辑

**42 天网格**:
```
上月末尾 (0-6 天) + 当月全部 (28-31 天) + 下月开头 = 42 天
```

**示例** (2026 年 2 月):
```
上月末尾：1 月 26-31 日 (6 天)
当月：2 月 1-28 日 (28 天)
下月开头：3 月 1-8 日 (8 天)
总计：6 + 28 + 8 = 42 天
```

## 🎨 交互流程

### 打开日历
```
用户点击日期标题
  ↓
setShowCalendar(true)
  ↓
显示日历弹出框
  ↓
点击遮罩层可关闭
```

### 选择日期
```
用户点击某个日期
  ↓
检查是否是未来日期
  ↓
  ├─ 是未来 → 无反应（禁用）
  └─ 非未来 → setSelectedDate(dateStr)
              ↓
              setShowCalendar(false)
              ↓
              自动加载该日期数据
```

### 切换月份
```
用户点击月份箭头
  ↓
更新 selectedDate（保持 1 号）
  ↓
日历重新生成
  ↓
显示新月份的日期
```
