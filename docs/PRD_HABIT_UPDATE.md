#### 4.5.3 习惯管理系统（更新版）

> **📅 更新日期**: 2026-03-01
> 
> **⚠️ 重要变更**: 废弃"自动关联规则引擎"设计，改为**独立习惯打卡 + Dashboard 快捷入口**模式。
> 
> **变更原因**:
> 1. 降低开发复杂度，聚焦核心价值
> 2. 避免功能冗余（用户无需重复操作）
> 3. 习惯定位为"坚持追踪"而非"数据记录"
> 4. Dashboard 热量 Card 已展示核心指标，无需重复

---

**核心设计原则**

1. **习惯定位**
   - 习惯打卡 = **二元完成型**（是/否，完成/未完成）
   - 计数累计型需求（如喝水杯数）在独立页面完成
   - 习惯管理页只做创建、汇总、分析，不直接打卡

2. **五个固定习惯**
   - 用户可选开通，开通后追踪完成情况
   - 每个习惯有独立打卡页面
   - Dashboard 提供快捷入口

3. **数据隔离**
   - 饮食/运动打卡 = 数据记录层（详细数据）
   - 习惯打卡 = 行为追踪层（完成状态）
   - 两者独立，不自动关联

**五个核心习惯**

| 序号 | 习惯名称 | 类型 | 打卡方式 | Dashboard 快捷入口 |
|------|---------|------|---------|-------------------|
| 1 | 每天保持热量赤字 | 自动计算 | 复用饮食 + 运动打卡 | ❌ 不显示（Card1 已展示） |
| 2 | 每天 8 杯水（2000ml） | 计数累计 | 独立饮水页面，点亮 8 个水杯 | ✅ 显示 |

| 4 | 每日早睡（23 点前） | 二元完成 | 独立早睡页面，睡前确认 | ✅ 显示 |
| 5 | 每日正念冥想 | 二元完成 | 独立冥想页面，确认完成 | ✅ 显示 |

**习惯 1：每天保持热量赤字**

| 维度 | 设计 |
|------|------|
| **定位** | 第一性原理，减重核心 |
| **打卡方式** | 复用现有饮食 + 运动打卡，自动计算 |
| **完成判定** | 摄入 < 消耗 = ✅ |
| **提醒设置** | 不单独提醒，跟随饮食/运动打卡提醒 |
| **Dashboard 展示** | ❌ 不显示快捷入口（Card1 热量净变化已展示） |
| **数据源** | 饮食打卡 + 运动打卡自动计算 |

**习惯 2：每天 8 杯水（2000ml）**

| 维度 | 设计 |
|------|------|
| **定位** | 提升代谢，最简单习惯 |
| **打卡方式** | 独立饮水页面，8 个水杯图标，用户自己点亮 |
| **完成判定** | 点亮 8 个水杯 = ✅ |
| **提醒设置** | 早、中、晚三次，显示目标完成进度 |
| **Dashboard 展示** | ✅ 快捷入口，显示进度（如 5/8 杯） |
| **打卡页面元素** | 8 个水杯图标、[+ 喝一杯] 按钮、进度条、喝水小贴士 |
| **科学话术** | "提升代谢 5-10%"、"减少虚假饥饿感"、"帮助排出代谢废物" |

**习惯 3：每天记录体重**

| 维度 | 设计 |
|------|------|
| **定位** | 追踪进展，数据反馈 |
| **打卡方式** | 独立体重页面，输入称重数据 |
| **打卡频率** | 建议晨起空腹，但支持一天多次记录 |
| **输入项** | 体重 (kg) 必填；测量时间（晨起/餐前/餐后/睡前）；可选体脂率、水分、肌肉含量 |
| **提醒设置** | 07:00 "晨起空腹称重最准确哦 ⚖️" |
| **Dashboard 展示** | ✅ 快捷入口，显示"未记录"或最新体重 |
| **打卡页面元素** | 体重输入框、测量时间单选、可选体脂数据、体重趋势图、称重小贴士 |
| **科学话术** | "晨起空腹最准确"、"固定时间看趋势"、"长期坚持者平均多减重 2.3kg" |
| **未来扩展** | 连接智能体脂秤自动同步 |

**习惯 4：每日早睡（23 点前）**

| 维度 | 设计 |
|------|------|
| **定位** | 降低皮质醇，提升减重效果 |
| **打卡方式** | 独立早睡页面，睡前确认 |
| **完成判定** | 用户主动打卡确认 23:00 前睡觉 = ✅ |
| **打卡内容** | 入睡时间、睡眠质量评分（1-5 星）、睡前准备 checklist |
| **提醒设置** | 21:30"准备洗漱"、22:00"放下手机"、22:30"调暗灯光"、23:00"晚安" |
| **Dashboard 展示** | ✅ 快捷入口，显示"未打卡"或"已完成✅" |
| **打卡页面元素** | 入睡时间输入、睡眠评分、睡前准备、本周睡眠统计、**每日情绪价值鸡汤** |
| **科学话术** | "睡眠不足 7 小时，第二天食欲增加 23%"、"皮质醇过高会让脂肪更容易堆积在腹部"、"深度睡眠时生长激素分泌，帮助燃脂" |
| **情绪价值示例** | "今天辛苦了，你值得一次优质的睡眠。放下一切，好好照顾自己。🌙" |

- [ ] 第 3 行快捷操作 4 个按钮（饮水、体重、早睡、冥想）
- [ ] 每个习惯有独立打卡页面
- [ ] 习惯管理页只显示汇总分析，不直接打卡
- [ ] 习惯开通流程显示价值说明
- [ ] 早睡打卡页面显示每日情绪价值鸡汤
- [ ] 冥想打卡页面显示每日灵性一句话
- [ ] 体重打卡支持一天多次记录
- [ ] 饮水打卡显示 8 个水杯图标和进度
- [ ] 提醒设置跟随具体习惯

**技术实现要点**

1. **Dashboard 布局**
   - 第 0 行：欢迎 Banner（动态文案 + 体重 + 坚持天数）
   - 第 1 行：热量净变化 Card（核心指标）
   - 第 2 行：运动打卡 + 饮食打卡
   - 第 3 行：4 个习惯快捷入口
   - 第 4 行：AI 健康顾问

2. **习惯打卡页面**
   - 饮水：8 个水杯图标，点亮逻辑
   - 体重：支持多次记录，趋势图
   - 早睡：入睡时间 + 睡眠评分 + 情绪鸡汤
   - 冥想：完成确认 + 灵性语录

3. **习惯管理页**
   - 已开启习惯列表（完成状态 + 连续天数）
   - 习惯分析（最佳/需加油/智能建议）
   - 习惯设置（提醒时间、目标调整）

4. **提醒系统**
   - 每个习惯独立提醒配置
   - 提醒内容动态生成（根据进度）
